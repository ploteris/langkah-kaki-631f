# Langkah Kaki - Landing Page

A modern, minimalist landing page for a local shoe brand built with Next.js, React, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.

## Features

- Hero section with product showcase
- Product collection with filtering
- About brand section
- Contact form
- Fully responsive design
- Smooth animations with Framer Motion
- Mobile-first approach

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion

## Design System

- **Colors**: Cream, Charcoal, Terracotta palette
- **Typography**: Inter (sans), Playfair Display (serif)
- **Animations**: Fade-in, slide-up, scroll-triggered
- **Spacing**: 5, 8, 12, 16, 20, 28, 32 (Tailwind scale)
